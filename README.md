#Using
clone pingxx plugin
    $git clone git@github.com:oumind/cordova-plugin-pingxx.git
    
clone the demo project
    $git clone git@github.com:oumind/cordova-pingxx-demo.git

Install the plugin
    $cd cordova-pingxx-demo
    $cordova plugin add ../cordova-plugin-pingxx/

Run the code
    $cordova run
